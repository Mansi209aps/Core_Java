# 🔥 Classes & Objects in Java

# 📌 1. Class Structure

## ✅ Theory

A **class** is a **blueprint/template** used to create objects.

👉 It defines:

* Variables (state)
* Methods (behavior)
* Constructors (initialization)

---

## 📌 Basic Syntax

```java
class ClassName {
    // variables (fields)
    // constructors
    // methods
}
```

---

## 📌 Complete Class Example

```java
class Car {
    String model;   // instance variable
    int speed;

    // constructor
    Car(String model, int speed) {
        this.model = model;
        this.speed = speed;
    }

    // method
    void display() {
        System.out.println(model + " " + speed);
    }
}
```

---

### 🔹 4. Blocks

A **block** is a group of statements enclosed within `{ }` (curly braces).
Blocks are used to **group code and control execution order**.


## 📌 Types of Blocks in Java

### 1️⃣ Static Block

#### ✅ Purpose:

* Used for **static initialization**
* Runs **only once** when class is loaded

### 📌 Code

```java
class Test {
    static {
        System.out.println("Static block");
    }

    public static void main(String[] args) {
        System.out.println("Main method");
    }
}
```

👉 Output:

```
Static block
Main method
```

## 2️⃣ Instance Initializer Block (Non-static block)

### ✅ Purpose:

* Runs **before constructor**
* Executes every time an object is created

### 📌 Code

```java
class Test {
    {
        System.out.println("Instance block");
    }

    Test() {
        System.out.println("Constructor");
    }

    public static void main(String[] args) {
        new Test();
    }
}
```

👉 Output:

```
Instance block
Constructor
```

## 3️⃣ Local Block (Normal block)

### ✅ Purpose:

* Used inside methods to limit scope of variables

### 📌 Code

```java
class Test {
    public static void main(String[] args) {
        {
            int x = 10;
            System.out.println(x);
        }
        // x cannot be accessed here
    }
}
```

### 📌 Execution Order (IMPORTANT)

```text
Static block → Instance block → Constructor → Method
```

# 📌 2. Object Creation

## ✅ Theory

An **object** is an **instance of a class**.

👉 Real-world:

* Class = Car blueprint
* Object = actual car

---

## 📌 Syntax

```java
ClassName obj = new ClassName();
```

---

## 📌 Example

```java
class Car {
    String model;

    void show() {
        System.out.println(model);
    }

    public static void main(String[] args) {
        Car c = new Car();
        c.model = "BMW";
        c.show();
    }
}
```

💡 Explanation:
Car c = new Car(); → creates an object

c.model = "BMW"; → assigns value to instance variable

c.show(); → calls method which prints model

---

## 📌 What happens internally? (VERY IMPORTANT)

```java
Car c = new Car();
```

### 👉 Steps:

1. Memory allocated in **heap**
2. Reference `c` created in **stack**
3. Constructor is called
4. Object initialized

---

# 📌 4. `this` Keyword (VERY IMPORTANT)

## ✅ Theory

`this` refers to **current object**

---

## 🔥 Uses of `this`

---

### 🔹 1. Resolve variable shadowing

```java
class Test {
    int x;

    Test(int x) {
        this.x = x;
    }
}
```

---

### 🔹 2. Call another constructor

```java
Test() {
    this(10);
}
```

---

### 🔹 3. Pass current object

```java
void show(Test t) {}

show(this);
```

---

### 🔹 4. Return current object

```java
Test get() {
    return this;
}
```

---

### 🔹 5. Call current class method

```java
void display() {
    this.show();
}
```
