# 🔥 2. Control Statements in Java

Control statements **control the flow of execution** in a program.


# 📌 Types of Control Statements

1. **Selection (Decision-Making)**
2. **Iteration (Looping)**
3. **Branching (Jump Statements)**

---

# 1️⃣ Selection Statements (Decision Making)

Used to **choose between multiple options**.

---

## 🔹 1. if Statement

```java
if (condition) {
    // code executes if condition is true
}
```

👉 Example:

```java
if (age > 18) {
    System.out.println("Eligible");
}
```

---

## 🔹 2. if-else Statement

```java
if (condition) {
    // true block
} else {
    // false block
}
```

---

## 🔹 3. if-else-if Ladder

Used for multiple conditions:

```java
if (marks > 90) {
    System.out.println("A");
} else if (marks > 75) {
    System.out.println("B");
} else {
    System.out.println("C");
}
```

---

## 🔹 4. Nested if

```java
if (a > 0) {
    if (a % 2 == 0) {
        System.out.println("Positive Even");
    }
}
```

---

## 🔹 5. switch Statement (VERY IMPORTANT)

Used for multiple fixed choices.

```java
switch (value) {
    case 1:
        // code
        break;
    case 2:
        // code
        break;
    default:
        // code
}
```

---

### 🔸 Important Points:

* Works with: `int`, `char`, `String`, `enum`
* `break` prevents fall-through
* `default` is optional

---

### 🔸 Fall-through Concept

```java
switch (x) {
    case 1:
    case 2:
        System.out.println("1 or 2");
        break;
}
```

---

### 🔸 Java 8+ String Support

```java
switch (name) {
    case "Mansi":
        System.out.println("Hello");
}
```

---

### 🔸 Java 14+ (Enhanced Switch - good to know)

```java
int result = switch(day) {
    case 1 -> 10;
    default -> 0;
};
```

---

# 2️⃣ Iteration Statements (Loops)

Used to **repeat code**.

---

## 🔹 1. for Loop

```java
for (initialization; condition; update) {
    // code
}
```

---

### Example:

```java
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
```

---

### 🔸 Variations (IMPORTANT)

#### Infinite for loop

```java
for (;;) {
}
```

#### Multiple variables

```java
for (int i = 0, j = 10; i < j; i++, j--) {
}
```

---

## 🔹 2. while Loop

```java
while (condition) {
    // code
}
```

👉 Executes **only if condition is true**

---

## 🔹 3. do-while Loop

```java
do {
    // code
} while (condition);
```

👉 Executes **at least once** (VERY IMPORTANT difference)

---

## 🔹 4. Enhanced for loop (for-each)

Used for arrays & collections:

```java
for (int num : arr) {
    System.out.println(num);
}
```

---

### 🔸 Limitations:

* No index access
* Cannot modify structure

---

# 3️⃣ Branching Statements (Jump Statements)

Used to **control loop flow**.

---

## 🔹 1. break

* Terminates loop or switch

```java
for (int i = 0; i < 10; i++) {
    if (i == 5) break;
}
```

---

### 🔸 Labeled break (IMPORTANT but less known)

```java
outer:
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        break outer;
    }
}
```

---

## 🔹 2. continue

* Skips current iteration

```java
for (int i = 0; i < 5; i++) {
    if (i == 2) continue;
    System.out.println(i);
}
```

---

### 🔸 Labeled continue

```java
outer:
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        continue outer;
    }
}
```

---

## 🔹 3. return

* Exits method completely

```java
return;
```

or

```java
return value;
```

---

# ⚠️ Important Edge Cases (Interview Traps)

---

### ❗ Difference: while vs do-while

| Feature   | while           | do-while      |
| --------- | --------------- | ------------- |
| Execution | 0 or more times | At least once |

---

### ❗ switch vs if-else

| switch       | if-else            |
| ------------ | ------------------ |
| Fixed values | Complex conditions |
| Faster       | More flexible      |

---

### ❗ break vs continue

| break     | continue       |
| --------- | -------------- |
| Exit loop | Skip iteration |

---

# 🎯 Interview Summary

If asked:

> “What are control statements?”

You say:

> Control statements in Java are used to control the flow of execution and are categorized into selection statements (if, switch), iteration statements (for, while, do-while), and branching statements (break, continue, return).

---

# ⚡ Pro Tips (VERY IMPORTANT)

* Always explain with **examples**
* Mention **edge cases** (like do-while runs once)
* Talk about **labeled break/continue** → gives extra points 💯
